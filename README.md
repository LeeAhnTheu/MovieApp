# Movie App
Small website for watching movie
## Clone
```
git clone https://github.com/doonamm/MovieApp
```
## First setup after cloning
```
npm install
composer install
```
copy .env file to root folder
## Run
```
php artisan serve
```
## Compile
```
npm run dev
```
## Watch change
```
npm run watch
```
